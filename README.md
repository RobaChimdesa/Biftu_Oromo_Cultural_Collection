# Biftu Oromo Cultural Collection

A modern multi-language Ethiopian cultural clothing e-commerce web application built with React, TypeScript, Tailwind CSS, and Framer Motion.  
This platform showcases authentic Oromo cultural clothes with elegant UI, animations, responsive layouts, and internationalization support.
